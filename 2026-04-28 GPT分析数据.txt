## 七、买入后上涨与下跌的决策树

### 买入后上涨

|   买入后收益 | 动作                      |
| ------: | ----------------------- |
|   0%-5% | 持有，不加仓                  |
|  5%-10% | 止损线抬到成本价                |

### 买入后下跌

|  买入后亏损 | 动作                                      |
| -----: | --------------------------------------- |
|  0%-3% | 观察，不补仓                                  |
|  3%-5% | 复核行业、估值、技术面，不补仓                         |
|  达到-5% | 进入浅套区，停止加仓                              |
|  达到-8% | 只有“逻辑未坏 + 技术企稳≥5项 + 估值更优”才允许补，且补仓≤原仓50% |
| 达到-15% | 不再摊薄，考虑换到更强资产                           |
| 超过-25% | 重大错误处理，止损换仓优先                           |

## 十、最终结论

你的组合当前不需要整体清仓，因为现金和稳健资产比例足够高，亏损仓也没有失控。
但你必须处理一个问题：**高浮盈过度集中在全球成长、先锋成长、全球优质这些高波动资产上。**

我的最终建议是：

**1. 节前不加仓。**
五一前不足5个交易日，不满足你的自检原则。

**2. 节前先锁定一部分利润。**
卖出 012920 的40%-50%，卖出 011892 的30%，018229达到+15%即卖30%。

**3. 亏损仓不补。**
有色、恒生科技、黄金都还没到必须补仓的阶段；跌到 -5%也不是立刻补，而是进入诊断。

**4. 节后第2-3个交易日再看买点。**
只有连续3日不创新低、MA5上穿MA10、站回MA20、成交放大这些条件至少满足2-3项，才考虑小额加仓。

**5. 正收益策略不是“保证市场上涨”，而是“先把已有利润落袋”。**
你现在有足够利润覆盖亏损，最专业的做法是把账面收益的一部分转成真实收益，然后再等待下一次更安全的买点。
